# ExamSystem
## 概述

## 安装
### 服务器要求
* PHP >= 7.0.0
* NodeJs >= 8.6.0
* Composer >= 1.5.2
* Laravel >= 5.5

### 安装依赖
#### 安装依赖的PHP包
`composer install`

#### 安装依赖的Node包
`npm install`

#### 编译、打包前端资源
`npm run environment`

## 运行
`php artisan serve`
